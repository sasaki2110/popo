### 文字列の操作
    明日はここからやね。
    スライス：ここ重要
        print(msg[1:3]) 
    ここの:3がなじまないなぁ・・・・1:3 で、[1][2]が表示されるって・・・
    先頭要素の位置　：　最終要素の位置＋１らしい。
    左を省略すると０から、右を省略するとlen(str)まで。
    ::step で取り出すステップも指定できる。

    in
    hasBook = "books" in "apple, book, car, bike"
    print(hasBook)

    split/join
    ar = "abc def ghi".split()
    print(ar)
    st = ",".join(ar)
    print(st)

    まあその他は必要都度でいいか。
